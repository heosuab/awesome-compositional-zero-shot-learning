## Compositional Zero-Shot Learning
[Compositional Zero-Shot Learning (CZSL)](https://paperswithcode.com/task/compositional-zero-shot-learning) is a computer vision task in which the goal is to recognize unseen compositions fromed from seen state and object during training. The key challenge in CZSL is the inherent entanglement between the state and object within the context of an image. <br>
Some example benchmarks for this task are MIT-states, UT-Zappos, and C-GQA. Models are usually evaluated with the Accuracy for both seen and unseen compositions, as well as their Harmonic Mean(HM).
